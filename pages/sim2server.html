You like the games such as the ones listed on the main page? Then the Sim2Server might be a good place for you.

Of course you can also just chill on that server (though you'd likely have more fun there if you like one or more of the games).

You can talk about the games, or also learn more about it or share your knowledge.

It doesn't matter how well you're experienced with the games, you can even be a newcomer to the games, or a master that can help out by questions or just be someone, that enjoys playing the games.

There are also some projects related about the games, such as Sim2Editor, which is a web browser save editor for Currently: The Sims 2 (Game Boy Advance & Nintendo DS), or also straight researches about the Savefiles with Sim2Research. There is also Optimum Alfred, a discord bot for Sim2Server which is still kinda a work in progress that should make the stay on that server more nicely.

You can find more information about the Server below.

Sim2Server Hub

Server Rules

  1. Don't break the discord's Terms of Service/Community Guidelines. (See here for the Community Guidelines)
  2. Don't talk or even discuss about piracy. That is a banned topic on this server.
  3. Be patient and polite. This is a friendly place, we don't need drama here.
  4. Try to stay on topic. That means -> try to use the proper channels.
  5. If you need to ask a question, directly ask the question and don't ask, if you can ask the question, because that only clutters the channels a lot more.
